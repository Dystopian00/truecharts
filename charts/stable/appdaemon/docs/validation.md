# Input Validation

**`LATITUDE`** and **`LONGITUDE`**
Accepted formats are:

- Float (eg. `24.2028`)

Regex used to match this: `^[0-9]{1,}\.{1}[0-9]{1,}$`
You can try live [here](https://regex101.com/r/xsLGWN/1)
