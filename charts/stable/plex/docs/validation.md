# Input Validation

**`Advertise IP`**
Accepted formats are:

- `https://192.168.1.100:32400`
- `https://192.168.1.100:32400`
- `http://dnsname:32400`
- `https://dnsname:32400`

Regex used to match this: `^http(s)?:\/\/.*:\d*$`
You can try live [here](https://regex101.com/r/3ehGWY/1)
